# Fenrir v0.4.0

> Framework for testing

## Install

### System requirements

* Unix-like OS
* `jq`
* `wget`
* `shellcheck`

### Installation

1. Clone this repository
2. Go to `src/main/fenrir/cli`
3. Run `./install-fenrir`

## Uninstall

* Clone this repository
* Go to `src/main/fenrir/cli`
* Run `./uninstall-fenrir`

## Usage

```
Possible options:
fenrir help
fenrir remote <URL>
fenrir task <TASK ID>
fenrir run <SOLUTION DIR>
------------------------------------------------------------------
fenrir help:
    Info about fenrir util
    See more https://gitlab.com/atpd/progtech/fenrir

fenrir remote <URL>:
    Set remote task repository

fenrir task <TASK ID>:
    Set task tests

fenrir run <SOLUTION DIR>:
    Run local tests on the specified local repository
```
